History & Japow

世界中が熱狂する
「JAPOW」

1996年にスイスで始まったFreeride World Tourは、2026年で30周年を迎え、出場選手数は1万人に達する規模へと拡大しました。

2017年に長野県白馬村での開催を皮切りに日本へ本格上陸。日本の極上のパウダースノー、通称「JAPOW（ジャパウ）」を滑走する映像は世界中に衝撃を与えました。

現在、FWT Japan Seriesは400名以上の国内外ライダーを誘致し、スノー業界で最も国際的な大会に成長しています。

Olympic Roadmap

冬季五輪競技化への
モメンタム

フリーライドは若い世代を中心に人気急上昇中のバックカントリー競技です。FWTの登録ライダー数は年間16.4%で成長しています。

FIS（国際スキー・スノーボード連盟）によるFWTの買収後、フリーライドは公式種目に採択されました。世界選手権の開催を通じて、オリンピック種目化に向けて大きく前進しています。

フリーライドは、日本人が得意とする緻密な操作性や独自のスタイルが活きる競技特性を持っており、日本選手陣の今後の活躍が多いに期待されます。

Criteria

01
Line Difficulty
SYS.01 // ラインの難易度

Difficulty of Line

ライダーが選んだラインのカッコよさ、オリジナリティを採点。クリエイティビティとイマジネーションが要求されるところです。ただし、リスクが高すぎるラインを選ぶと減点になります。

02
Control
SYS.02 // コントロール

Control

いかにカッコいいラインを選んでも、コントロールされた滑りが出来なければ、得点は高くなりません。転倒したり、バランスを崩したりすると、大きく減点されます。事前の斜面チェックでの見積もりの精度が重要になります。

03
Fluidity
SYS.03 // 流動性

Fluidity

ストップ・アンド・ゴーを繰り返さず、自分の選んだラインを、スタートからゴールまで一気に滑り降りているかどうかを採点します。横移動が長すぎたり、登って逆戻りしたり、ジャンプの前に何秒も停止したりすると減点されます。

04
Jumps
SYS.04 // ジャンプ

Jumps

ジャンプは、フリーライドの大会で非常に重要な要素です。クールなスタイルとアグレッシブなジャンプは大きく加点されます。大きさ、入り方、空中でのトリック、着地、全体的なコントロールが採点の基準になります。

05
Technique
SYS.05 // 技術

Technique

アルペンスキーや基礎スキーで言われるスキーの技術のことです。ライダーの技術不足が原因でコントロールを失ったとき、このセクションが減点されます。他の選手がターンを刻んでいた箇所で横滑りを行ったりすることも減点対象です。
